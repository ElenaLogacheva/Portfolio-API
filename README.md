# Portfolio-API
Используя fetch и FakeStoreAPI (https://fakestoreapi.com/docs) получите описание 10 продуктов.

Затем отобразите результат запроса в виде карточек товара, где будет фотография товара, цена и название. Все необходимые поля для этого есть в объекте товара, который приходит с сервера. Должно выглядеть примерно так:
